Danang Ade Hidayat - UI/UX Designer

Peran: UI/UX Designer

Project: Website Ecommerce Rebook

Periode: 1 Maret 2026 - 10 Juni 2026

Jobdesk & Task (Maret–Juni 2026)

Maret 2026 — Riset

  • Kumpulkan requirement: fitur inti (browse buku, pencarian, filter, detail buku, keranjang, checkout, pembayaran, status pesanan)
  • Audit kompetitor & referensi (Tokopedia/Bukalapak, Gramedia, marketplace buku bekas)
  • Riset pengguna:
  • Susun rencana riset + skrip interview/survey
  • Interview calon pengguna (pembeli & penjual) + catat pain points
  • Sintesis: affinity mapping, insight utama
  • Buat persona & user journey (pembeli & penjual)
  • Definisikan problem statement + prioritas (MoSCoW)
  • Susun user flow high-level: login/daftar, pencarian–pembelian, jual buku (jika ada), tracking order
  • Buat sitemap / information architecture

April 2026 — Wireframe & Validasi

  • Home, kategori, search & filter
  • Detail buku (foto, kondisi, deskripsi, seller info)
  • Keranjang, checkout, alamat, metode pembayaran
  • Buat low-fi wireframe untuk halaman utama:
  • Login/daftar, profil, riwayat pesanan
  • Desain state & empty state (kosong, error, loading)
  • Review internal dengan stakeholder/dev (feasibility)
  • Prototyping low-fi (klikable)
  • Usability testing (5–8 partisipan) + catat isu
  • Iterasi wireframe berdasarkan feedback
  • Grid/spacing, typography, color awal
  • Mulai definisi design system awal:
  • Komponen dasar: button, input, search bar, card buku, badge kondisi
  • Output akhir bulan: wireframe tervalidasi + daftar perbaikan + komponen dasar

Mei 2026 — UI High-Fidelity & Design System

  • Buat moodboard & arah visual (brand Rebook)
  • Desain UI high-fi untuk flow utama:
  • Browse → detail → add to cart → checkout → pembayaran → konfirmasi
  • Color tokens, typography scale, elevation/shadow, radius
  • Definisikan design system lebih lengkap:
  • Komponen: navbar, tab/filter, modal, toast, pagination, stepper checkout
  • Susun guideline: penggunaan komponen + do/don’t
  • Prototyping high-fi (interaksi utama, microcopy)
  • Accessibility check (kontras, ukuran font, fokus, label)
  • Spesifikasi ukuran, spacing, state, dan asset (ikon/gambar)
  • Handoff awal ke developer:
  • Catatan perilaku komponen & rules validasi form
  • Output akhir bulan: UI high-fi + design system v1 + prototype high-fi

Juni 2026 — Handoff, QA, dan Iterasi

  • Finalisasi file desain (rapikan naming, struktur halaman, komponen reusable)
  • Buat dokumentasi handoff:
  • User flow final, spesifikasi komponen, dan acceptance criteria per halaman
  • Support developer selama implementasi (klarifikasi & revisi minor)
  • Responsiveness (mobile/tablet/desktop jika scope)
  • State/error/empty/loading
  • QA UI (cek implementasi vs desain):
  • Konsistensi typography, spacing, warna
  • UAT/feedback stakeholder + log issue
  • Iterasi berdasarkan temuan QA/UAT
  • Output akhir bulan: desain final + daftar issue/bug UI + revisi pasca QA
